# Publicar la web

- **Local:** `npm run dev` → http://localhost:3000

## Opción fácil: Vercel (recomendado)

1. Sube el proyecto a GitHub (repo **ana-maria-medina** o el que sea).
2. Entra en **https://vercel.com**, inicia sesión con GitHub.
3. **Add New → Project** → importa el repo.
4. En configuración: **Output Directory** = `out`. Deploy.
5. URL: **https://ana-maria-medina.vercel.app** (o la que te den).

Más opciones (Netlify, Cloudflare, GitHub): **DESPLIEGUE-GRATIS.md**.
